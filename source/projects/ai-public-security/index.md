<main class="detail-content">
<section id="overview">
  <h2>项目概述</h2>
  <p>本项目用于记录算法、平台、数据系统和公共安全治理之间的关系，特别关注技术应用背后的组织流程和责任结构。</p>
</section>

<section id="questions">
  <h2>研究问题</h2>
  <ul>
    <li>人工智能系统如何改变公共安全治理中的信息收集与决策过程？</li>
    <li>技术系统引入后，组织责任和风险承担如何重新分配？</li>
    <li>公共安全治理中的效率、合法性和可解释性如何平衡？</li>
  </ul>
</section>

<section id="data">
  <h2>数据与材料</h2>
  <ul>
    <li>公共安全、智慧治理和人工智能相关政策文本。</li>
    <li>应用场景案例、公开招投标文件和系统说明材料。</li>
    <li>相关法律法规、伦理规范和治理评估资料。</li>
  </ul>
</section>

<section id="methods">
  <h2>方法与工具</h2>
  <p>可采用政策文本分析、案例研究和技术治理文献综述。材料整理阶段可建立应用场景和治理风险分类表。</p>
</section>

<section id="progress">
  <h2>阶段进展</h2>
  <p>当前阶段先聚焦概念和案例收集，形成公共安全领域人工智能应用场景清单。</p>
</section>

<section id="plans">
  <h2>后续计划</h2>
  <ul>
    <li>整理人工智能与公共安全治理的核心文献。</li>
    <li>收集地方智慧治理案例。</li>
    <li>设计技术应用场景的比较表。</li>
  </ul>
</section>

<section id="notes">
  <h2>相关笔记</h2>
  <p>后续可以链接算法治理、风险治理和公共安全政策笔记。</p>
</section>
